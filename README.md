# 📑 Side Bar Menu (CSS-Only Navigation)
> Um componente de navegação lateral (Sidebar) moderno, totalmente responsivo e construído exclusivamente com HTML5 e CSS3, eliminando a necessidade de JavaScript para sua funcionalidade principal.

## 📖 Sobre o Projeto
Este projeto faz parte do meu portfólio de estudos em Programação Full-Stack. O objetivo principal foi desenvolver uma barra de navegação lateral fixa com suporte a ícones e redes sociais, focada em performance e acessibilidade.

O diferencial técnico desta implementação é o uso de um `input type="checkbox"` para controlar a abertura e o fechamento do menu. Esta abordagem garante que a interface seja reativa e funcional mesmo em ambientes onde o JavaScript possa estar desabilitado ou ainda não tenha sido carregado.

## ✨ Destaques e Aprendizados Técnicos
* **Gerenciamento de Estado Nativo:** Uso de `<input type="checkbox">` e <label> para alternar a visibilidade da barra lateral de forma declarativa.

* **Interface Rica em Ícones:** Integração com a biblioteca Font Awesome (v5.15.2) para fornecer dicas visuais claras em cada item do menu, como Dashboard, Atalhos, Serviços e Contato.

* **Arquitetura de Navegação:** Estruturação semântica utilizando listas não ordenadas `(<ul>)` e links de âncora para garantir uma boa base para SEO e acessibilidade.

* **Engajamento Social:** Inclusão de uma seção dedicada a ícones de redes sociais (Facebook, Twitter, Instagram e YouTube), essencial para aplicações modernas de impacto na comunidade.

* **Separação de Responsabilidades:** Organização clara do projeto com o arquivo de estilos isolado em um diretório próprio (css/style.css), seguindo as melhores práticas de engenharia de software.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica e lógica de interface via inputs.

* **CSS3:** Estilização de layout, posicionamento fixo e animações de transição.

* **Font Awesome:** Biblioteca de ícones vetoriais para suporte visual.
